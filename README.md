# Project-1-Client-Query_Management_System

Client and Query Management System - The web application is to manage the queries between Client and Support.
Built with help of using Python, Streamlit and SQLite .

FEATURE 

→CLIENT MODULE:

   ⁕Submit a new query

   ⁕Enter email, mobile number, heading, and description

   ⁕Query automatically stored with created time

   ⁕Status set as Open by default

   ⁕View all submitted queries

→SUPPORT MODULE:

   ⁕View all client queries

   ⁕Filter by All, Open, and Closed

   ⁕Add closing time when query resolved

→DATABASE(SQLite)

   Two tables are created:

1. Users

     ⁕username

     ⁕hashed_password

     ⁕role

2.Queries

   ⁕id (auto generate)

   ⁕email

   ⁕mobile

   ⁕heading

   ⁕description

   ⁕created_time

   ⁕closed_time

   ⁕status

→TECH STACK:

   ⁕Python

   ⁕Streamlit(Frontend + Backend)

   ⁕SQLite3(Database)

   ⁕Datetime(Timestamps)

   ⁕Hashlip(Password hashing)
